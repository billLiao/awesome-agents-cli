## Featured Categories

- **🤖 AI Coding** - Aider, Cursor, Continue
- **💬 Chat & LLM** - Ollama, Llama.cpp, GPT4All
- **⚡ Automation** - NanoBot, Open Interpreter, Devika
- **🛠️ Dev Tools** - GitHub Copilot CLI, ShellGPT
- **📚 Frameworks** - LangChain, LlamaIndex, AutoGen

## Quick Start

```bash
# Install Ollama for local LLMs
curl -fsSL https://ollama.com/install.sh | sh

# Install Aider for AI pair programming
pip install aider-chat

# Install Open Interpreter for task automation
pip install open-interpreter
```
