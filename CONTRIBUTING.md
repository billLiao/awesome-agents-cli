# Contributing to Awesome Agents CLI

Thank you for your interest in contributing! This project aims to curate the best CLI-based AI agents and tools.

## How to Contribute

### Adding a New Tool

1. **Find the right category** - Place your tool in the most appropriate section
2. **Follow the format**:
   ```markdown
   ### [Tool Name](https://url.com) ⭐
   Brief description of what it does.
   ```bash
   # Installation command
   command to install
   ```
   ```
3. **Include essential info**:
   - Clear description
   - Installation command
   - Usage example
   - Badges for key features (🆓 free, 🔥 popular, etc.)

### Categories

- **AI Coding Assistants** - Tools that help write/edit code
- **AI Chat & Conversation** - LLM runners and chat interfaces
- **Task Automation** - Autonomous agents and task runners
- **Development Tools** - Dev-focused AI utilities
- **System & Productivity** - General productivity tools
- **Framework & SDKs** - Libraries for building AI apps

### Quality Standards

- ✅ Must have CLI interface
- ✅ Must be actively maintained
- ✅ Should have clear documentation
- ✅ Preferably open source or has free tier

### Review Process

1. Submit your PR with changes
2. Maintainer will review within 48 hours
3. Once approved, it will be merged

## Code of Conduct

- Be respectful and inclusive
- Provide constructive feedback
- Help maintain quality standards

## Questions?

Open an issue for any questions or suggestions.
